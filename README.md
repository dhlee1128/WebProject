# WebProject
Spring 기초 학습을 위한 프로젝트로 간단한 회원가입 페이지를 만들고 MariaDB와 연동하였습니다.
개념 정리를 위한 것으로 중요한 기술을 적용해 보는 것과 Test 코드를 작성하는 것을 중점으로 진행하였습니다.


## 개발 환경
* VSCode 
* Java 11
* Gradle 7.5
* Spring Boot 2.7.2
  
## 기술 스택
* Spring Boot
* Spring Web
* Thymeleaf
* Spring Boot DevTools
* MySQL Driver
* Jdbc Template
* Spring Data JPA
* AOP
